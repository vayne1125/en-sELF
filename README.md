## Execute project
### cmd
```
cd C:\Users\user\AppData\Local\Android\Sdk\emulator
emulator -list-avds   //see the phone's name you create
emulator -avd xxx     //xxx is your phone name
```
### vscode
```  
cd enUsELF
npx react-native run-android
```
